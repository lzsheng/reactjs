
## react.js-base项目

> - 基于
> - react.js - v15.x
> - react-router - v3.x
> - webpack - v1.x
> - es6
> - sass

### 安装&运行


```
git clone https://github.com/lzsheng/reactjs.git
```
然后进入项目路径base下：


```
cd ./base
```

安装依赖包


```
npm install
```

本地运行


```
npm run startDevHot
```
毫无意外的话，打开浏览器，输入url访问

```
localhost:8008
```
